`by default`

- inline:
	- can't change weight or height
	- stays in same line with other elements
- inline-block:
	- can change weight and height
	- stays in same line with other elements
- block: 
	- can change
	- doesn't stay in same line as other

we set it using `display` key, the same one we use to change
