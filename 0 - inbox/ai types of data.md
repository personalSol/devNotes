A data is mainly of two types based on how it is stored:
- structured data: data in tables, csv, etc
- unstructured data: data in the form of images, vidoes, audios, transcripion of video, etc are unstructured data

Based on nature of data, it is of two types:
- Static data: Data which is fixed. Data in excel like name, age, etc. 
- Streaming data: data which is ever/constantly changing over time. Data like news, etc
