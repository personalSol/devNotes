| Modifier    | Class | Derived Class | Outside |
| ----------- | ----- | ------------- | ------- |
| `public`    | ✅     | ✅             | ✅       |
| `protected` | ✅     | ✅             | ❌       |
| `private`   | ✅     | ❌             | ❌       |
- **`public`**: Accessible everywhere.
- **`protected`**: Accessible in class & subclasses/child class.
- **`private`**: Accessible only within the class.

**Default**:

- `class` members → `private`
- `struct` members → `public`
